add_sile¶

sisl.io.add_sile(suffix, cls, case=True, gzip=False)¶

Add files to the global lookup table

Public for attaching lookup tables for allowing users to attach files externally.

Parameters

  • suffix (str) – The file-name suffix, it can be several file endings (.TBT.nc)

  • cls (child of BaseSile) – An object that is associated with the respective file. It must be inherited from BaseSile.

  • case (bool, optional) – Whether case sensitivity is applicable for determining file.

  • gzip (bool, optional) – Whether files with .gz endings can be read. This option should only be given to files with ASCII text output.
